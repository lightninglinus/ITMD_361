Course Catalog Description:

This course will cover the use and building blocks of websites by using HTML, CSS, Javascript, and graphical applications. Networked multimedia distribution technologies are also explored. The design of effective Websites include page layout, user interface design, graphic design, content flow and site structure as well as management of web site resources including intranet management and design considerations are addressed. Students design and create a major website with multiple pages and crossed-linked structures. Prerequisites: [none] Credit: 2-2-3 Semester Hours

Course Outcomes:

Students completing this course will be able to:

  • Recognize HTML, CSS and JavaScript mark-up and code.
  • Select the proper mark-up tags or code to achieve a particular result.
  • Identify improperly used mark-up and code.
  • Produce modern standards complaint web pages.
  • Deploy web pages to a public server.

Course Objectives:

At the conclusion of this course, successful students will be able to:

  • Explain the client and server architecture of the internet and related web technologies.
  • Use a basic text editor and other software tools to create web pages using HTML, CSS, JavaScript, jQuery, and images.
  • Deploy web pages to a Gethub using version control.
